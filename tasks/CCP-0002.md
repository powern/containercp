# CCP-0002: Generate .env file

## Description

The provider must generate a `.env` file inside the site directory
containing environment variables required by the Docker Compose stack.

## Required variables

```
SITE_DOMAIN=<domain>
SITE_OWNER=<owner>
MYSQL_DATABASE=<sanitized domain>
MYSQL_USER=<owner>
MYSQL_PASSWORD=<auto-generated>
MYSQL_ROOT_PASSWORD=<auto-generated>
```

## Acceptance criteria

1. `.env` exists after `site create`
2. Passwords are generated randomly
3. Variables are referenced by `docker-compose.yml`
4. Re-running create fails — env file is not overwritten
5. `.env` is listed in `.gitignore` (if present)
