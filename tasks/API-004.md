# API-004: GET /api/jobs

## Description

List background jobs with status and progress.

## Requirements

GET /api/jobs

Response:
```json
{
    "success": true,
    "data": [
        {
            "id": 1,
            "type": "site_create",
            "status": "running",
            "progress": 60,
            "steps": ["Validating", "Creating filesystem", "Creating database"],
            "current_step": 2,
            "created_at": "..."
        }
    ]
}
```
