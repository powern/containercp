# API-003: POST /api/backups/create

## Description

Add REST endpoint to create a backup via the backend.

## Requirements

POST /api/backups/create

Body:
```json
{"domain": "example.com"}
```

Response:
```json
{"success": true, "data": {"filename": "example.com-20240101.tar.gz", "size": 1234}}
```

Uses TarBackupProvider internally.
