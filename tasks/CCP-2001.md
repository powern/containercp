# CCP-2001: Introduce shared utility module

## Description

Password generation logic is duplicated in `EnvGenerator::generate_password()`
and `SiteCreateOperation::generate_password()`. Extract into a shared module
that can be used across the project.

## Requirements

Create:

libs/utils/
    Random.h
    Random.cpp
    PasswordGenerator.h
    PasswordGenerator.cpp

Namespace:

containercp::utils

### PasswordGenerator

```
static std::string generate(int length);
```

Default length: 32

Allowed characters: A-Z, a-z, 0-9

Uses std::random_device and std::mt19937.

### Removal

Replace all inline password generation with `PasswordGenerator::generate()`.

Remove `SiteCreateOperation::generate_password()`.

Update `EnvGenerator` to use the shared utility.

## Acceptance criteria

1. PasswordGenerator produces 32-char random strings
2. All callers use the shared utility
3. No duplicate password code remains
