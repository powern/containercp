# CCP-0003: Harden docker-compose.yml

## Description

The generated `docker-compose.yml` must include production-ready
settings for reliability and resource management.

## Required changes

### Restart policies

Every service must have:

```yaml
restart: unless-stopped
```

### Health checks

- nginx: HTTP health check on port 80
- php: TCP health check on port 9000
- mariadb: SQL health check
- redis: Redis ping health check

### Resource limits

```yaml
deploy:
  resources:
    limits:
      memory: 256M
    reservations:
      memory: 128M
```

## Acceptance criteria

1. All four services have `restart: unless-stopped`
2. All four services have a health check
3. Memory limits are set for each service
4. Stack starts without warnings
5. `docker compose ps` shows healthy services
