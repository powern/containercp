# JOB-001: In-memory job tracking

## Description

Create a simple job tracking system for background operations.

## Requirements

### Job struct

```
- id
- type (string: "site_create", "backup", "restore", etc.)
- status (string: "pending", "running", "completed", "failed")
- progress (int: 0-100)
- steps (vector of step descriptions)
- current_step (int)
- message (string)
- created_at (string)
- completed_at (string)
```

### JobManager

```
- create(type, steps)
- update(id, status, progress, message)
- find(id)
- list()
- next_id()
```

### Location

libs/jobs/Job.h, Job.cpp, JobManager.h, JobManager.cpp

### Integration

- ServiceRegistry owns JobManager
- DaemonApp can create and update jobs
- API can list and query jobs
