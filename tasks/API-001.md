# API-001: POST /api/sites/create

## Description

Add REST endpoint to create a site via the backend.

## Requirements

### Endpoint

POST /api/sites/create

Body (JSON):
```json
{
    "owner": "admin",
    "domain": "example.com"
}
```

Response:
```json
{
    "success": true,
    "data": {
        "domain": "example.com",
        "message": "Site created"
    }
}
```

### Implementation

- Parse JSON body
- Call SiteCreateOperation via daemon
- Return the result

### Router update

Add POST route support. The Router currently only matches exact
method+path. POST routes need to be registered separately.
