# Tech Stack

## Language

ContainerCP is written in C++.

Target standard:

- C++20

## Build system

- CMake
- Ninja

## Compiler

Primary compiler:

- g++

Secondary compiler:

- clang++

## First milestone

The first milestone is a CLI-only MVP.

No web UI yet.

## Design preference

Prefer:

- small modules
- simple interfaces
- minimal dependencies
- clear error messages
- safe filesystem operations
- explicit confirmation before destructive actions
