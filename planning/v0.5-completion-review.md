# Sprint Review: Version 0.5 Completion

## Implemented

### API layer (ARCH-001)
- POST /api/jobs?id=N — job detail endpoint with progress, steps, status
- POST /api/domains/remove — remove domain by fqdn
- POST /api/databases/remove — remove database by name
- POST /api/ssl/remove — remove SSL certificate
- POST /api/proxy/remove — remove proxy config
- POST /api/access-users/remove — remove access user
- POST /api/ssl/enable — enable SSL certificate
- POST /api/ssl/disable — disable SSL certificate
- Unified remove_resource and modify_ssl helper lambdas reduce code duplication

- Dashboard with recent jobs list and live count loading

- Form validation (owner required, domain format)
- All CRUD actions call real backend API endpoints
